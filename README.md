# 🐦 Tappy Bird

A simple Flappy Bird-inspired arcade game created by **Siddimaster (MCO)** using **Scratch 3.0** and packaged with **TurboWarp**.

## 🎮 About

Tappy Bird is a lightweight endless arcade game where players control a bird and navigate through obstacles while trying to achieve the highest score possible.

Built in Scratch 3.0 and exported using TurboWarp for improved performance and standalone distribution.

## ✨ Features

- Simple one-button gameplay
- Endless obstacle generation
- Score tracking
- Fast and lightweight
- Built with Scratch 3.0
- Packaged using TurboWarp

## 🕹️ Controls

| Action | Control |
|----------|----------|
| Fly Up | Spacebar / Click |
| Start Game | Spacebar / Click |
| Restart | Spacebar / Click |

## 🛠️ Built With

- Scratch 3.0
- TurboWarp Packager

## 📦 Packaging

This project was packaged using TurboWarp to provide:

- Better performance
- Standalone gameplay
- Faster loading times
- Improved compatibility
